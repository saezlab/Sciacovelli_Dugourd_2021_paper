# Sciacovelli_Dugourd_2021_paper

code of theanalysis for the Sciacovelli_Dugourd_2021 manuscript.

Scripts are located in the scripts folder. 

In order to run them, simply open Sciacovelli_Dugourd_2021_paper.Rproj in R studio.

You will need to instal the following packages:

From bioconductor:
library(GSEABase)
library(biomaRt)
library(clusterProfiler)
library(dorothea)
library(fgsea)
library(limma)
library(viper)
library(vsn)

From Cran:
library(cowplot)
library(dplyr)
library(ggfortify)
library(ggplot2)
library(ggrepel)
library(grid)
library(gridExtra)
library(igraph)
library(pheatmap)
library(readr)
library(readxl)
library(reshape2)
library(visNetwork)

From github:
library(decoupleR) (https://github.com/saezlab/decoupleR)
library(ocean) (https://github.com/saezlab/ocean)

Enjoy !
